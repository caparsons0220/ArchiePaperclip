# Archie Web

New Archie Bravo frontend app.

Purpose:

- customer-facing Archie Bravo product shell
- talks to the existing Paperclip backend over `/api`
- replaces the current Paperclip UI for end users
- leaves `ui/` available as the native/internal admin surface

Suggested first scaffolding targets:

- `src/app/`
- `src/components/`
- `src/features/`
- `src/lib/`
- `public/`

Recommended first pages:

- `/`
- `/workspaces`
- `/workspace/[workspaceId]`
- `/agenda`
- `/journal`
- `/settings`
